Первым делом устанавливаем CUDA https://developer.nvidia.com/cuda-downloads
Затем выходим в папку с проектом, в командной строке запускаем: pip install -r requirements.txt
После чего проект можно запустить с помощью: python manage.py runserver

Первый запуск будет долгим в случае если не установлена сама модель StableDiffusion (тогда она сначала будет скачана). Если же веса модели уже установлены,
в pipeline.py исправляем model_address на локальный путь до модели.

Последующие запуски тратят немного времени на инициализацию пайплайна (до 10 секунд), но в остальном время занимает только сама генерация.
